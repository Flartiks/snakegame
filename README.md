# How to build?
Dependences - cmake, git, g++ / MinGW 

On linux - `Cmake -B build`
then - `Cmake --build build`

If you use mingw(Minimalist GNU for Windows):
`Cmake -B build -G "MinGW Makefiles"`
then - `Cmake --build build`
